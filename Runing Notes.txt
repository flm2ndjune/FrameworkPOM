Feature: OrangeHRM Login
  As a user of OrangeHRM
  I want to login to the system
  So that I can access my dashboard securely

  Background:
    Given I am on the OrangeHRM login page

  @ValidLogin
  Scenario: Successful login with valid credentials
    When I enter username "Admin"
    And I enter password "admin123"
    And I click the Login button
    Then I should be redirected to the dashboard
    And I should see the user profile icon

  @InvalidLogin
  Scenario: Login with invalid username
    When I enter username "WrongUser"
    And I enter password "admin123"
    And I click the Login button
    Then I should see an error message "Invalid credentials"

  @InvalidLogin
  Scenario: Login with invalid password
    When I enter username "Admin"
    And I enter password "wrongPass"
    And I click the Login button
    Then I should see an error message "Invalid credentials"

  @EmptyFields
  Scenario: Attempt login with empty fields
    When I click the Login button
    Then I should see validation messages for username and password

  @EmptyFields
  Scenario: Attempt login with only username
    When I enter username "Admin"
    And I leave the password field empty
    And I click the Login button
    Then I should see a validation message for password

  @EmptyFields
  Scenario: Attempt login with only password
    When I leave the username field empty
    And I enter password "admin123"
    And I click the Login button
    Then I should see a validation message for username

  @ForgotPassword
  Scenario: Verify Forgot Password link
    When I click on the "Forgot your password?" link
    Then I should be redirected to the password reset page

  @Logout
  Scenario: Successful logout after login
    When I enter username "Admin"
    And I enter password "admin123"
    And I click the Login button
    And I click the Logout link
    Then I should be redirected to the login page

  @Security
  Scenario: Access dashboard without login
    Given I navigate directly to the dashboard URL
    Then I should be redirected to the login page

Feature: Hotel reservation date validation

  Scenario: Verify that check-out date cannot be earlier than check-in date
    Given the user launches the hotel reservation application "http://adactinhotelapp.com/"
    And the user logs in with username "{test username}" and password "{test password}"
    When the user selects location "Sydney"
    And the user selects hotel "Hotel Creek"
    And the user selects room type "Standard"
    And the user selects number of rooms "1"
    And the user enters check-in date as "today + 7 days"
    And the user enters check-out date as "today + 5 days"
    And the user clicks on the search button
    Then the system should display an error message "Check-In date should not be later than Check-Out date"
